How to Use

  1. Enter a website URL in the input field (include http:// or https://)
  2. Click the "Get HTML" button to fetch the HTML code
  3. View the syntax-highlighted HTML code in the results section
  4. Use the "Copy" button to copy the code to your clipboard
  5. Use "Toggle Wrap" to enable/disable text wrapping

This version uses a Node.js backend server, which helps bypass CORS restrictions and provides better error handling than client-side only solutions.
